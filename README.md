# Car Rental Management System
Made For Our Group's OOPT Assignment.

**To Get Started :** Run The "MainManager.java" File.

**Notes :**
1. System Data Can Be Found In The ".txt" Files Located Within "src/data/".
2. Please Ensure Your Machine Has A Minimum Of ***JDK 17*** Or Higher To Support Pattern Matching.
3. For Quick Testing Purposes, 3 Default Users Have Been Prepared With Simple IDs And Passwords:
   - Admin: A0001, adminpass
   - Staff: S0002, staffpass
   - Customer: C0003, custpass
